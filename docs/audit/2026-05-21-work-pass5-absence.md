## Stage Contract

### Purpose
誰沒有出聲？什麼解法 / 立場 / 視角在這 15 篇 + top reply 之間沒有承載？這是 pipeline 中**唯一允許推論性結論**的 pass，但每個結論必須掛 evidence-strength 標記。

### Evidence-strength 標記（本 pass 的核心紀律）
每個 absence claim 標三級之一：

- **[強]** — 缺席在 evidence 中是 conspicuous 的：present 的東西強烈反襯 absent 的東西（例：所有人都在算帳，但沒有人算「集體議價」）。這級接近可斷言
- **[中]** — 缺席成立但可能是 sample artifact：只有 15 篇、只有 top-3 reply，無法排除「其實有，只是不在樣本 / 不在 top reply」
- **[弱/推論]** — 純推論，需要更多資料才能確認。明確標為 speculative

### 三種不同強度的「缺席」主張（不可混用）
1. **「不在 top-3 reply」** — 最弱，只代表讀者投票結果，thread 內可能有
2. **「不在這 15 篇 + 其 top reply」** — 中等，是本 sample 的缺席
3. **「不在這個 discourse / reality」** — 最強，本 pass 原則上**不做這級斷言**，除非有跨 pass 的強 converging evidence，且即使如此也標 [弱/推論]

### What this pass does NOT do
- 不重寫 narrative（P3）/ audience（P4）
- 不寫 final editorial（P6）
- 不引用 Codex northstar
- 不把「缺席」直接等同於「應該存在」——缺席是觀察，不是規範主張

## 1. 缺席的說話者：被討論但從不發聲的人

這個 topic 有大量「被當作物件討論、但從不作為主體發聲」的角色。

### 1.1 僱主 / 定價方 [強]
僱主在多篇被討論：`S9.OP` 罵僱主用低薪要求年輕人、`S10.OP` 的「公司潛規則」、`S13.OP` 招聘 asso admin 開 16-20k。但**真正設定工資、設定潛規則的僱主從不發聲**。最接近的是 `S13.R2`（招聘者承認 anti-master degree bias）——但這條 reply 的功能是**佐證學歷貶值**，不是辯護定價邏輯。沒有任何聲音解釋「為什麼 admin 開 16-20k」「為什麼要請食糖水」。
**反襯證據：** 整個 topic 充滿對僱主行為的算帳（P2 §1.3），但算帳的人全部是被定價方，定價方零發聲。

### 1.2 被指為工資壓力來源的群體 [強]
`S9.OP/R1` 把外勞 / 新移民 / 菲傭印傭當作工資壓低的 agent，`S12.R1` 提「內地護士來港」。這些群體被精確命名（甚至有工資比例「1/3-1/4」`S9.R1`），但**他們本人從不發聲**——是純粹的 object。`S8.OP` 的「大媽 / 長者 / 全職主婦」同樣是被供養的 object，零主體性。
**反襯證據：** P4 §2.4 reader 對 OP 的 self-report 有 high prima facie credibility——但這個 credibility 不延伸到被指涉的第三方，第三方連 self-report 的機會都沒有。

### 1.3 被討論職業的真正從業者 [中]
keeta 司機（`S5.R1`、`S12.OP`）、護士（`S12`）、殯儀館看更（`S15`）、admin（`S13`）——這些職業全部被當作「飽和 / 不值得 / 最後退路」討論，但**沒有任何一篇是這些職業的從業者本人發聲**。`S12.R1` 是醫生（相鄰職業）不是護士本人。
**為什麼是 [中] 而非 [強]：** 這是 15 篇的採集結果，無法排除「Threads 上其實有護士 / 看更發聲，只是沒被採進這個 topic」。缺席是 sample-level 的。

### 1.4 親密關係中被隱瞞的對象 [中]
`S12.OP`「唔敢同老婆講」——老婆是被隱瞞的對象，從不發聲。`S15.OP` 的母親是少數**間接發聲**的家人（「老母講笑咁俾我睇」），但仍是經 OP 轉述，且 queued 無留言。
**觀察：** 家人作為角色，分成「被隱瞞」（S12 老婆）vs「遞送 grim 訊息」（S15 母親）兩種，但都沒有獨立 voice。

### 1.5 成功兌現投入的同代人 [強]
P3 N1「投入不再兌現」的核心是 sunk-cost。但**沒有任何訊號描述「投入有兌現」的可信同代人**——唯一的 `S6.OP`（創業成功）被自我 self-undercut + `S6.R2` 諷刺為繼承（P3 N1 inversion）。
**反襯證據：** P2 §6.4「整個 audit 的擴張詞集中在唯一被質疑的訊號」+ P4 §2.3「沒有 reader 為 escape ramp 辯護」——成功的可信版本在 OP 端和 reader 端**雙重缺席**。

## 2. 缺席的立場

### 2.1 「集體悲觀其實未必準確」的反方 [強]
整個 topic 沒有任何聲音質疑「飽和 / 貶值」是否被誇大。`S5.R1`「廢水而家開始變剛性需求咁 無話氾唔氾濫」是唯一接近的——但這是業界 insider 反駁單一行業的供過於求預設，不是反駁整體悲觀。
**反襯證據：** P2 §1.1「飽和」橫跨 4 個 sector 失去 sector specificity，但**沒有任何 reply 質疑這個詞的過度使用**。P4 §2.4 reader 對 OP self-report 的 high credibility 延伸到「悲觀 report」——悲觀有 prima facie 可信度，樂觀沒有。

### 2.2 僱主 / 制度方的自我辯護 [強]
見 §1.1。沒有「定價是合理的」「潛規則有其功能」「招聘 bias 有其理由」的立場。

### 2.3 為某條 escape ramp 辯護的立場 [強]
P4 §2.3 已記錄：所有 escape ramp 的 reader 動作清一色是 disqualification 或內部分裂，**沒有 unconditional defense**。P3 N2「出路同步自證偽」的另一面就是「沒有人替出路辯護」。
**反襯證據：** `S5.R1` 的 conditional endorsement（如果你要做就這樣做）是唯一接近的，但它本身以「先諗可以蝕幾多」（`S5.R2`）收結。

### 2.4 質疑 OP 是否誠實 / 是否過度戲劇化 [中]
P4 §2.4：除 `S6.R2` 質疑 attribution（資金來源），沒有 reader 質疑其他 OP 的 self-report 真實性。
**為什麼是 [中]：** 這是 top-3 reply 的觀察，thread 內可能有質疑但沒進 top 3。但「沒進 top 3」本身是訊號——讀者群不願意 upvote 質疑受苦者的 reply。

## 3. 缺席的解法

### 3.1 集體 / 制度層 prescription [強]
P4 §2.2 + P3 N4 已強力 converging：individual-level prescription 出現（`S5.R1`、`S10.R2`），但**沒有任何「行業協會 / 政府 / 工會 / 同行 / 學校 / 媒體應該做 X」的主張**——即使在結構性命名最清晰的 `S9`、`S11`、`S8` 的 top reply 內也沒有。
**反襯證據：** P4 §1.4 reader 願意把 OP 升級成結構分析（`S9.R1` 519 字、`S11.R2` feedback loop）——**讀者有能力 analyze，但 analysis 從不轉成 prescription**。診斷力與處方力之間有斷層。

### 3.2 集體行動 vocabulary [強]
P2 §6.2：工會 / 罷工 / 集體議價 / 修法 / 申訴 / 勞工法 全部零出現。
**反襯證據：** `S10` 的「過試用請食糖水」是明顯的集體議題（30 人公司的共同潛規則），但 reader 給的是 individual 解（請 / 不請），**沒有人說「呢個 practice 整個行業應該取消」**。集體議題被 individual 化處理。

### 3.3 跨代理解的正面案例 [中]
`S1`（90後 vs Gen Z）、`S6`（80後）、`S9`（後生仔 vs 老一輩）都觸及世代，但**全部是隔閡 / 對立，沒有跨代理解的範例**。`S1.R1`「90尾你都係 GenZ」是消解差異不是理解差異。
**為什麼是 [中]：** 15 篇樣本的世代議題本來就少，難以斷言「跨代理解不存在於 discourse」。

## 5. 結構性缺席：消失的中間層（本 pass 最強推論）

> **[強] —— 這是 P5 的核心結論，建立在 4 個 pass 的 converging evidence 上**

把前面所有缺席收攏，浮出一個 single structural absence：

**這個 discourse 有兩個 scale 的完整 vocabulary，但中間層完全空白。**

- **個體 scale：** 算帳（17k vs 1000 蚊）、self-discount（唔叻 / 太弱）、微反抗（薪水小偷）、隱瞞（唔敢同老婆講）——vocabulary 豐富（P2 §1.2-1.3）
- **社會 / 結構 scale：** 飽和、AI 取代、外勞政策、稅收結構、惡性循環——vocabulary 也豐富（P2 §1.1、P3 N4、P4 §1.4）
- **中間層（collective / intermediate scale）：** 行業協會、工會、同行串連、世代群體、社區、媒體評論、政黨——**vocabulary、speaker、solution 三者全部缺席**

### 為什麼這是 [強] 而不是 [中]
四個 pass 的 evidence 獨立 converge 到同一個 gap：
1. **詞層**（P2 §6.2）：集體 / 制度 vocabulary 零出現
2. **敘事層**（P3 N4）：結構性指認 + individual advice 並存，中間完全空
3. **audience 層**（P4 §2.2）：reader 只給 individual prescription，零 collective prescription
4. **audience 結構層**（P4 §2.5）：top reply authors 沒有任何重複——**連 reader pool 本身都不形成 community**。每個 thread 是 isolated audience

第 4 點是最有力的：缺席不只在「內容」，**也鏡像在 discourse 進行的方式本身**。15 個 atomized threads、non-repeating participants——這個 discourse 的物理結構就是 individual ↔ system 的直接對撞，沒有經過任何 collective intermediary。**缺席的中間層不是話題選擇問題，是這個 platform discourse 的結構特徵**。

### 這個推論的邊界
- **不能斷言** 香港社會沒有中間層組織（工會、協會確實存在於 reality）
- **只能斷言** 在這 15 篇 Threads 訊號 + 其 top reply 構成的 discourse 裡，中間層的 vocabulary / speaker / solution 缺席，且這個缺席在四個分析層獨立成立
- **無法排除** 這是 Threads platform 的 affordance 造成（短帖、匿名、演算法推送個人化內容）而非議題本質——這個 alternative explanation 必須在 P6 / 跨 topic 對照中處理

## 6. Outlier / boundary 作為缺席的反證

### 6.1 S4：在場卻缺席的 work 議題
`S4`（外國 vs 香港 share screen + 廁所笑話）在 topic 內，但 P3 確認它在四條敘事都不在、P4 確認它是 topic drift。**它的「在場」反而標記了 topic 邊界**——自動採集會把「提到工作場景」的訊號收進來，但內容對議題零貢獻。S4 的存在揭露的不是一條缺席的聲音，而是 **topic 邊界本身需要被 audit 識別**。
**Validator hook：** S4 的 Spoiler 圖不可得，drift 是 reader-driven 還是 visual-driven 不可判定。

### 6.2 S6：被壓制的成功敘事
`S6` 是唯一的「投入有兌現」訊號（P3 N1 inversion）。它的存在 + 被雙重壓制（self-undercut + `S6.R2` 諷刺）**證明成功敘事不是「沒人講」，而是「講了不被信」**。這比單純缺席更強——缺席可能是沒人想講，但 S6 顯示的是：講了，但這個 register 在當前 discourse 沒有 credibility。**這是 active suppression 不是 passive absence**。
