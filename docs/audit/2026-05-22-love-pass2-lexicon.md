## 1. 反覆出現的詞群

### 1.1 市場 / 評分 / 算帳 詞群（密度極高）
戀愛被量化的詞彙橫跨多篇：
- 評分人：「7 至 9 分」「1-5 分」「top 5% 條件」「1/50」`S2.OP`；「10 個港姐冠軍」「驚世美女」`S2.OP`
- 供需 / 演算法：「競爭池」「挑剔模式」「驚艷模式」「無限供應」「機會率同性價比」「niche market」`S2.OP`
- 資產清單：「有車有樓」「工作穩定」「月薪幾多」「幾時供完樓」「身家」`S3.OP`
- 收入軌跡：「月入 6 萬」「由萬 9 人工變 10 幾萬」`S7.OP`

這組詞與 work 的算帳詞群同源（work 是 17k / 50-70 萬 / 裁員百分比），但**被搬進戀愛場**——擇偶被當成市場定位、男人被用收入編號。`S3.OP` 甚至明說 app 自介「好似見工咁」——直接把戀愛比作求職。

### 1.2 平台 / App 詞群
「swipe」「match」「profile」「users」「演算法」「del app」「已退 apps」`S1/S2/S9`——平台動作詞。注意 exit 詞：「已退 apps」`S1.R1`、「想 del app」`S9.OP`。

### 1.3 正面 / 關係詞群（**work 完全缺，love 有**）
- 甜蜜 / 儀式：「勁開心」「彈下彈下咁跑過黎」「嘩嘩聲」「接人收工仲開心過俾人接」`S6.OP`
- 承諾 / 未來：「執子之手，與子偕老」「白首不離」`S6.R1`；「未來另一半」「我同老公係喺交友 app 識」`S4.R1`
- 幸福：「對方有回應，就係幸福」`S6.R2`；「我算幾幸福」`S4.R1`

這組詞在 work 的 P2 §6 被列為「完全缺席」（work 無「希望 / 前景 / 幸福」）。**love 有，且帶 future-positive 時態（與子偕老 / 未來另一半）**——詞層紀錄，cross-topic 對照留後。

### 1.4 認命 / 挫敗 / 退場 詞群
「要認命」`S7.OP`、「懷疑人生」`S3.OP`、「好灰想 del app」`S9.OP`、「冇幾個正常」`S9.OP`、「認真嘅人真係會玩到好炆」`S9.R1`。這組與 work 收縮詞群（飽和 / 唔敢）功能相近，但 love 的退場是**離開平台 / 認命**，不是 work 的「自我降權後繼續困在原地」。

### 1.5 戲謔 / 粗俗 / dark humor 詞群
「墊塊 m 巾」`S5.OP`、「玩一條 JJ / 玩十條 JJ」`S8.OP`、「師兄好忙」`S8.R2`、「左右都係男人」`S8.R1`。與 work 戲謔詞群同功能（把脆弱降壓成可說），但 love 的戲謔更身體化 / 性化。

### 1.6 規範 / 教化 詞群（**值得特別記錄**）
- 「靚女唔應該上 App」「唔靚嘅先應該玩 App」「應該盡量喺現實生活」`S2.OP`——明確 prescriptive
- 「奉勸女孩子們」`S5.OP`——勸世語
- 「Put yourself in his shoes，我地一齊諗下」`S4.OP`——引導反思
- 「叫我拍拖」`S8.OP`（母親）——他人規範

這組詞 work 幾乎沒有（work P2 §4.2「命令式只出現於否定方向，沒有 prescriptive direction」）。**love 有正向 prescription（應該做 X，不只「唔好做 X」）**——詞層紀錄。

## 2. OP-coined vs Reader-coined

### 2.1 OP-coined
- 「溝仔無數嘅靚女姐姐」「挑剔模式 / 驚艷模式」`S2.OP`（自鑄市場理論術語）
- 「好似見工咁」`S3.OP`（戀愛=求職的比喻）
- 「墊塊 m 巾」`S5.OP`（戀愛腦的身體化 tip）
- 「玩一條 JJ / 十條 JJ」`S8.OP`
- 「我必須非常努力才顯得毫不費力」`S9.OP`（自嘲式引用）

### 2.2 Reader-coined / reader 提供的 reframe
- 「心態決定行為」「你個心想搵咩人，就會吸到咩人」`S4.R1`（用心理學 reframe 反駁 OP）
- 「受資本影響（香港女性通病）」`S7.R2`（用 ideology 詞 reframe OP 的價值觀）
- 「你一直係陪人由低谷行到高處嘅人，只係未遇啱」`S7.R1`（therapeutic reframe「認命」）
- 「自卑到自大」被 `S9.R1` 引為共鳴錨點（其實是 OP 先講，reader 接）

**Pattern 差異（vs work）：** work 的 reader-coined 詞集中在「去神聖化」（高級興趣班 / Permanent head Damage）。love 的 reader-coined 集中在 **reframe OP 的心態 / 價值觀**（心態決定行為 / 受資本影響）——reader 在 love 更傾向**校正 OP 的認知框架**，而非補完 OP 的省略。

## 3. 同詞 / 關鍵詞的多向使用

### 3.1 「心態」
- `S1.OP`「咩心態」——OP 用來評判他人（已讀不回的人）
- `S4.R1`「心態決定行為…你個心想搵咩人」——reader 用來把責任拉回 OP 自己

同一個詞，OP 用它指向他人，reader 用它指向 OP——**「心態」是 love 裏分配責任的關鍵詞**。

### 3.2 「正常」
- `S9.OP`「冇幾個正常」「終於識到個正常人啦」+ `S9.R1`「App 入邊真係無個正常」——「正常」是 love 裏的稀缺欲求對象，如同 work 裏的「兌現」。

### 3.3 「回應 / response」
- `S1.OP` 已讀不回（無回應）；`S6.R2`「對方有回應，就係幸福」；OP handle `no_responsesss`——「回應」橫跨負面（S1）與正面（S6），是 love 的核心稀缺品。

### 3.4 「分 / 評分」
- `S2.OP` 把人評分（7-9 分 / 1-5 分），且自承「靚唔靚係相對的」——OP 既用評分系統又承認它不穩定。

## 4. 句式 / 短語模式

### 4.1 Numbered thesis（S2 獨有）
`S2.OP` 用 1-10 點 numbered 結構建構理論——與 work `S11.OP`（AI 裁員 6 點）同為 numbered，但 S2 是**規範性立論**（應該怎樣），S11 是**描述性羅列**（發生了什麼）。

### 4.2 反問 + 反諷
「你係咪搞錯咗？」「不如直接寫埋月薪幾多」`S3.OP`、「你做咗幾耐呀咁，一年！」`S9.OP`——OP 用反問表達不滿。

### 4.3 Self-resignation 收尾
「嗯，我知道我係要認命」`S7.OP`、「好灰想 del app😔」`S9.OP`——以放棄 / 退場收尾（對照 work 的 self-rhetorical question 收尾）。love 較多直接 resignation，work 較多開放提問。

### 4.4 母親引語
「媽咪上個星期同我講叫我拍拖」`S8.OP`——母親的話被引述後被 OP 戲謔擋回（對照 work S15 母親遞送殯儀館看更）。

## 6. 詞層缺席（word-level）+ work 發現的逐項測試

### 6.1 測試 work 發現①：算帳 vocabulary —— **love 同樣有，甚至更尖銳**
work 的「算帳是 default cognitive mode」在 love **成立且加強**：S2 把人評分、S3 把戀愛比求職、S7 用收入編號男友。**戀愛被市場化的詞彙密度不低於 work**。→ 算帳詞群**不是 work 特有**，跨兩個 topic 都在。

### 6.2 測試 work 發現②：future-tense / 正面詞 缺席 —— **love 破功（有）**
work 的「無 future tense、希望零出現」在 love **不成立**：S6「與子偕老 / 白首不離」、S4.R1「未來另一半 / 我算幾幸福」——love 有 future-positive vocabulary。→ 「無 future tense」**是 work 議題特性，不是 Threads platform affordance**（初步詞層證據，P5 / cross-topic 再確認）。

### 6.3 測試 work 發現③：prescriptive direction 缺席 —— **love 破功（有）**
work 的「命令式只在否定方向，沒有 prescriptive direction」在 love **不成立**：S2「靚女應該 / 唔靚應該」、S5「奉勸」、S4「我地一齊諗下」——love 有正向 prescription。→ prescription 缺席**也是 work 特性**。但注意：love 的 prescription 全是 **individual-strategy scale**（個人擇偶策略），不是集體 / 制度——這點與 work 一致。

### 6.4 測試 work 發現④：集體 / 制度 vocabulary 缺席 —— **love 同樣缺**
love 完全沒有「集體 / 共同 / 一起組織 / 制度改變」的 vocabulary。**有結構性對象詞**（「演算法存在惡意」`S2.OP`、「受資本影響」`S7.R2`）但與 work 一樣**停在指認，沒有集體回應**。→ 集體 vocabulary 缺席**跨兩個 topic 都在**——這是 work 發現中最可能是 platform affordance 的一項。

### 6.5 love 自己的詞層缺席
- 沒有「結婚 / 生育 / 長期規劃」的務實詞（除 S6 的 emotional 承諾）——戀愛被講成 app 體驗 / 情緒事件，較少 institutional（婚姻登記 / 經濟結合 / 家庭）詞
- 沒有男性第一人稱發聲的詞彙標記——9 篇 OP 多為女性視角（S2 靚女姐姐 / S7 香港女性 / S9 女性遇 gym-bro），S8 性別曖昧
