## Stage Contract

### Purpose
從 15 篇 EvidencePackets + Pass 1 readings 自由歸納**詞彙層**的觀察。focus 是 word / phrase level，不是 narrative、不是 reaction 模式、不是位置缺席。

### Output
- 詞群（自由命名，不限數量）
- OP-coined vs reader-coined coinage 分佈
- 同詞反向使用
- 短語模式 / 句式特徵
- 語言 register（code-mixing、平台/地名/機構詞庫）
- 詞層缺席（word-level only：哪些**詞**沒出現，不是哪些**立場**沒出現——後者在 P5）

### What this pass does NOT do
- 不命名 narrative cluster（P3）
- 不命名 audience reaction 模式（P4）
- 不做 speaker / position 缺席（P5）
- 不打 enum / 不存 schema field
- 不做摘要式 vocabulary count（lexicon 的價值在 reframe，不在 frequency）

### Citation
所有詞彙引用必須註明出處 `S#.OP` 或 `S#.R#`。同一個詞橫跨多篇就全部列。

## 1. 反覆出現的詞群

### 1.1 飽和 / 供過於求 / 氾濫（整 audit 最密的詞群）

**最高密度的詞家族**：「飽和」、「供過於求」、「氾濫」三個近義詞在 15 篇橫跨 5 個訊號出現：

- 「以家已經供過於求、氾濫左」`S5.OP`
- 「keeta 又飽和」`S12.OP`
- 「千祈唔好讀護士，現在已經飽和」`S12.R1`（醫生身份補刀）
- 「廢水而家開始變剛性需求咁 無話氾唔氾濫」`S5.R1`（業界 insider 反駁）
- 隱含於 `S13.OP`「收到超過40個申請有碩士學位」（碩士供過於求）

跨 5 個訊號 + 反方反駁同詞 + 多行業同詞使用，這是整個 audit 詞彙場中**最有共識性的詞群**。三個近義詞共同說的事：**個體不是被某事件擊敗，是被結構性過剩擠壓**——但「結構性過剩」這個書面詞本身在 15 篇中沒有出現，全部用粵語 vernacular 替代。

### 1.2 收縮 / 不敢 / 自卑 詞群

- 「唔敢同老婆講」`S12.OP`
- 「敢唔敢做」`S15.OP`（母親說 / OP 重複）
- 「驚開口同同事溝通都驚驚青青」`S2.OP`
- 「好驚自己做錯，又好驚俾人覺得我『露晒餡』」`S2.OP`
- 「25歲人先嚟懷疑人生，係咪我太弱？」`S2.OP`
- 「自卑」`S2.OP`
- 「本身自己又唔係叻」`S10.OP`
- 「就係融入唔到」`S1.OP`

這組詞的共同功能不是描述事件，是**先 disqualify 自己的能動性**。S2 + S10 + S12 + S15 + S1 五個訊號的 OP 至少有一句明確的 self-discount——OP 在描述困境前先給自己降權。值得注意 `S12.OP` 35 字 OP 把「唔敢」用在配偶溝通上，不是工作溝通——把職場困境 escalate 為私人關係的隱瞞。

### 1.3 算帳 / 量化詞群

15 篇中精確數字異常密集：

- `S10.OP`：17k 月薪 + 1000蚊 + 500蚊 + 30人
- `S13.OP`：16-20k + 40個申請
- `S15.OP`：30k 月薪
- `S5.R1`：50-70萬開業 + 16-18k 全職 + 55-60/hr 兼職 + 25-30% 材料 + 28-30% Keeta 抽 + 12-15% 利潤 + 15-20% 冬天跌幅
- `S8.OP`：200萬交稅人口 + 1/3 比例
- `S11.OP`：15% 員工 + 8000 人 ×2（渣打 + Meta） + 4年 + 9年前 + 「每星期3日，每日4小時」
- `S6.OP`：七年 + Bentley + 啟德 + 西沙

這組詞反映的不是富裕，是**所有發聲者都在算帳**——失業者算供樓（`S12.OP`），新人算糖水（`S10.OP`），預想創業者算 franchise 成本（`S5.R1`），時評者算裁員百分比（`S11.OP`），納稅人算被供養比例（`S8.OP`）。**「算帳」是這個 topic 的默認 cognitive mode**。即使 `S6.OP` 的勵志訊號也用 Bentley + 啟德 + 西沙等具體資產列舉，不用抽象成就詞。

### 1.4 戲謔 / 粗口 / 自嘲詞群

- 「粗口」`S14.OP`（用作 PhD 完成感想，名詞化）
- 「仆你街，洗唔洗俾埋個屎忽你吊丫，都On9嘅」`S9.OP`（粗口開場）
- 「叫佢地食屎啦」`S10.R1`（2200 likes，全篇 audit 第二高 reply）
- 「死窮鬼」`S10.OP`（自稱）
- 「薪水小偷行動升級」`S3.OP`（自鑄身份）
- 「全民整水喉」`S11.OP`（諷刺）
- 「我靜左」`S14.OP`（極簡白描的戲劇靜默）
- 「夢裏一定什麼都有」`S6.OP`（self-undercut 結尾）

戲謔的功能可分兩種：(a) **dark humor 包裝困境**（S10、S14、S6）；(b) **粗口作為憤怒釋放**（S9.OP、S10.R1）。沒有一篇是純粹娛樂性戲謔——所有玩笑都掛在困境上。`S14.R1`「Permanent head Damage 你估簡單呀，沒有讀書就沒有傷害」用 PhD 縮寫雙關，是這組詞最精煉的 case。

### 1.5 潛規則 / 職場儀式詞群

- 「公司潛規則」`S10.OP`
- 「試用期」`S10.OP`
- 「請大家食糖水/飲廢水」`S10.OP`
- 「人情壓力」（隱含於 `S10.OP` 的「有意無意都會講『你就嚟過試用喇喎，有咩表示？』」）

## 2. OP-coined vs Reader-coined 詞

詞彙的源頭區分很重要——哪些是 OP 帶來的、哪些是讀者反過來給 OP 的。

### 2.1 OP-coined（OP 自鑄）
- 「薪水小偷行動升級」`S3.OP`
- 「養起無數大媽同長者」`S8.OP`
- 「後生仔做遮醜布」`S9.OP`
- 「死窮鬼」`S10.OP`（自稱）
- 「全民整水喉」`S11.OP`
- 「唔敢同老婆講」`S12.OP`
- 「敢唔敢做」`S15.OP`（從母親處接收）

### 2.2 Reader-coined（讀者反過來給 OP 的詞）
- 「高級興趣班」`S13.R1`（309 likes，給碩士的去神聖化詞）
- 「唔夠一年嘅興趣班」`S13.R2`（243 likes，繼承 R1 並具體化為 taught master）
- 「Permanent head Damage」`S14.R1`（281 likes，給 PhD 的雙關）
- 「廢水而家開始變剛性需求」`S5.R1`（42 likes，給「廢水」反向 framing）
- 「叫佢地食屎啦」`S10.R1`（2200 likes，給潛規則的拒絕詞）
- 「先諗每個月可以蝕幾多」`S5.R2`（4 likes，給創業者的反向格言）
- 「整水喉都冇得做」`S11.R2`（32 likes，把 OP 的諷刺升級為 feedback loop）
- 「減人工」`S9.R2`（56 likes，把 OP 怒氣壓縮成三字結論）

**Pattern 觀察**：reader-coined 詞集中在學歷 / PhD / 創業 / 結構性裁員四個主題——這幾個主題的 reader 比 OP 更會找精煉詞。`S13`、`S14` 兩篇是 reader-coined 詞 likes 達到三位數的訊號（309、243、281），且 OP 本身的詞量極少（`S13.OP` 一句、`S14.OP` 四句）——**OP 越省略，reader 越用 coined 詞補位**。

### 2.3 reader 順 OP 詞跟風但沒升級的訊號
- `S3.R1/R2` 接「薪水小偷」往 protocol 升級（「飲多啲水加速排毒」、「去廁所記得去耐啲」）但沒有自鑄詞
- `S11.R1`「睇緊點考水電A牌」順 OP 的「全民整水喉」punchline 但沒升級
- `S8.R1`（pinned）「養起一個全職主婦」延伸 OP「養起大媽長者」

### 2.4 distribution
- **沒有 reader-coined 詞的訊號**：S1 / S2 / S4 / S6 / S7 / S8 / S9 / S10 / S12 / S15（10 篇）
- **有 reader-coined 詞的訊號**：S3（OP-coined 為主，reader 接力）/ S5 / S11 / S13 / S14（5 篇）

整體上 OP-coined 多於 reader-coined（7 vs 8，數量相當但 OP-coined 跨 7 篇，reader-coined 集中在 5 篇）。**最有精煉力的詞集中在 S13 + S14 兩篇**——學歷貶值與 PhD 倦怠是讀者最願意參與 coining 的主題。

## 3. 同詞反向用法

### 3.1 「廢水」
- `S10.OP`「公司潛規則係要請大家食糖水/飲廢水」——**廢水 = 被迫消費的廉價飲品 + 職場稅項物**
- `S5.R1`「廢水而家開始變剛性需求咁 無話氾唔氾濫」——**廢水 = 反潮流的市場機會 + 必需品**

同一個 vernacular 詞在兩個 reader 共識中完全反向：S10 的讀者承認廢水是負擔（R1 2200 likes endorse「叫佢地食屎啦」），S5.R1 反過來說廢水變剛需。這不是矛盾，是兩個 reader 都從**「廢水」承載的廉價 + 大眾化**屬性出發，只是判讀方向不同。

### 3.2 「飽和」
- `S5.OP`「以家已經供過於求、氾濫左」——OP 用作**自身（茶飲）行業**的描述
- `S12.OP`「keeta 又飽和」——OP 用作**他人行業**的描述（外送平台）
- `S12.R1`「現在已經飽和」——醫生用作**護士行業**的描述（橫向擴展）
- `S13.OP` 隱含「碩士供過於求」——招聘者用作**人才供應**的描述

「飽和」橫跨四個不同 sector，沒有任何一個 reply 質疑這個詞的有效性。這個詞**已經失去 sector specificity，變成 generic 形容詞**。

### 3.3 「興趣班」
- `S13.R1`「讀master ... 一向係高級興趣班」——general master degree
- `S13.R2`「Taught master ... 唔夠一年嘅興趣班」——具體類型 taught master

兩個 reader 接力把同一個 reframe 詞細分粒度。R1 提出 reframe，R2 補上「不夠一年」的具體 disqualifier。

## 4. 句式 / 短語模式

### 4.1 Self-rhetorical question 收尾

15 篇中至少 5 篇 OP 用 self-rhetorical 句收尾：

- 「係咪我太弱？真係好想即刻走人...」`S2.OP`
- 「唔應該再考慮？」`S5.OP`
- 「究竟我應該要點做好？」`S10.OP`
- 「而家去讀護士好唔好...」`S12.OP`
- 「敢唔敢做」`S15.OP`

這個句式不期待答案，但**把 cognitive 負擔轉給 reader**——OP 沒結論但向讀者開放。對應的 reader 反應差異很大：S2 給 emotional permission（不答）、S5 給經濟拆解（不答原問題，反駁前提）、S10 給兩種互相矛盾的策略（同時答）、S12 用 credential 否決（反向答）、S15 暫無 reply。**同一個句式 + 不同 reader 反應 = 5 種不同互動模式**——這個觀察留給 P4。

### 4.2 Modal verb of inhibition：唔敢 / 唔好 / 千祈唔好

- 「唔敢同老婆講」`S12.OP`、「敢唔敢做」`S15.OP`：OP 表達自身 inhibition
- 「大家唔好再讀碩士啦」`S13.OP`：OP 命令式禁止
- 「千祈唔好讀護士」`S12.R1`：reader 命令式禁止
- 「唔好諗每個月賺幾多？」`S5.R2`：reader 格言式 reframe

整個 lexicon 場中**命令式句頻繁出現於否定方向**——「不要做 X」遠多於「應該做 X」。這跟 §1.2 的 self-discount 詞群形成呼應：個體層面 self-disqualify，集體層面 prohibition——但沒有 prescriptive 方向。

### 4.3 OP 自己列舉現實的 numbered structure

只有 `S11.OP` 用 6 點 numbered 寫成短時評——其他 OP 都是 prose 或單句。這個 structure 跟 OP 的 data-driven 內容（具體 corporate 名 + 數字）匹配。在 15 篇 OP 中是格式異類。

### 4.4 Hashtag 列表（只有 S9）

`S9.OP` 結尾「#年輕也是有罪 #將心比己啦唔該 #你哋啲仔女都受緊 #所以咪無人生bb囉 #又要馬兒好又要馬兒不吃草」——hashtag 做了 prose 沒做的事：**把職場議題明確接到生育議題**。其他 14 篇 OP 都沒用 hashtag。

## 6. 詞層缺席（word-level only）

P2 只列**詞層**缺席，不討論 speaker / position 缺席（後者在 P5）。

### 6.1 沒有出現的擴張 / 上升 vocabulary

15 篇 OP + top replies 沒有出現的詞（這些詞在 HK 職場語境中是常見的，缺席因此值得記錄）：

- **「上車」**、「翻身」、「致富」、「發達」—— HK 經典 aspirational vocabulary
- **「升職」、「加人工」、「跳槽」（positive sense）** —— 唯一相關的「轉第二份工」`S10.OP` 是在自我 disqualify「又未必會好似而家呢度咁穩定」
- **「機會」、「賺到」、「轉型成功」、「翻盤」** —— career-positive vocabulary
- **「主動」、「議價」、「爭取」、「talk salary」** —— agency vocabulary

唯一接近擴張詞的是 `S6.OP`：「夢想」、「目標」、「為夢想奮鬥」、「肯努力去試」——但這篇本身帶 self-undercut（「夢裏一定什麼都有」）和讀者諷刺（`S6.R2` 質疑資金來源）。**整個 audit 的擴張詞集中在唯一被質疑的訊號中**，是一個強烈的 lexical signal。

### 6.2 沒有出現的集體 / 制度 vocabulary

- **「工會」、「集體議價」、「罷工」、「申訴」、「勞工法」、「條例」、「最低工資」（明確）** —— collective / institutional response vocabulary
- **「修法」、「政策提案」、「投訴」、「申冤」、「行政上訴」** —— civic / procedural vocabulary
- `S9.R1` 提到「政府冇規劃好嘅位置」「最重要個職位規管」——這是最接近 institutional 詞的留言，但屬於**指認**而非**主張**

### 6.3 沒有出現的 future-positive vocabulary

- **「希望」** —— 不出現於任何 OP 或 top reply
- **「期待」、「展望」、「前景」（positive sense）** —— 不出現
- 「未來」只在 `S11.OP`「未來 4 年全球中後台炒 15%」這個負向 framing 出現
- 「以後」、「下一步」、「new path」 —— 不出現

時間軸詞彙嚴重偏向過去（「辛苦左咁多年」`S14.OP`、「終於完成」`S14.OP`、「9年前預言」`S11.OP`、「七年了」`S6.OP`）和威脅性未來（`S11.OP` 的裁員預言）。**沒有中性或正面前景描述的時間詞彙**。

### 6.4 沒有出現的「成功」描述詞

`S6.OP` 是唯一勵志訊號，但他的成功描述是**列舉資產**（Bentley、啟德天璽、西沙），不是描述能力、技能、知識、聲譽、社會貢獻——也就是**沒有 craftsmanship vocabulary**。即使勵志訊號裡，成功也只能用資產量化，不能用 intrinsic value 描述。

### 6.5 邊界注意

P2 不主張「這些詞缺席等於這些立場缺席」。**詞層的缺席只說明這個語言空間目前不流通某些詞**，不直接等於立場不存在。P5 absence pass 會處理 position-level 的缺席（誰沒發聲、什麼解法沒出現），P2 的職責止於詞層紀錄。

## 7. Hook notes for P3 / P4 / P5

P2 自然浮出但**不在 P2 內解讀**的觀察，移交下游：

### → P3 narrative
- 「飽和」橫跨 5 個訊號 + 4 個 sector 但沒有 sector specificity——這個詞群可能是某條 narrative 的骨架
- 「算帳」作為 default cognitive mode 可能撐起一條 narrative
- 自我鎖死 + 命令式禁止 + 沒有 prescriptive 方向 = 某種敘事結構

### → P4 audience reaction
