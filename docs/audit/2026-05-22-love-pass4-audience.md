## 1. 觀察到的反應型態

### 1.1 校正 OP 的認知框架 / ideology（love 顯著，work 罕見）
Reader 不回應 OP 的處境，而是 attack OP 看事情的框架本身。
- `S7.R2` luffy_623（63）：「點解你用收入去衡量一個男士的價值？…太受資本影響（香港女性的通病）」——指認 OP 的衡量標準是意識形態
- `S4.R1` karrie.in.london（130）：「心態決定行為！你個心想搵咩人，就會吸到咩人」——把 OP 的場域論（app 沒好人）reframe 為心態論
- `S7.R1` mathkinger（190）：「唔係你要認命，係你遇到嘅人都有自己課題…未遇啱」——reframe「認命」為「未遇啱」

**對照 work：** work 的 reader 接 OP 的 frame（即使反方向也同 frame），love 的 reader **改 OP 的 frame**。這是兩個 topic 最不同的 audience 動作。

### 1.2 雙向反駁同一 OP（love 獨有）
- `S4`：`S4.R1` 樂觀反駁（app 可行，我同老公就係 app 識）+ `S4.R2` cynical 反駁（app 上好條件者係掠食者）——同一 thesis 被兩個相反方向拆。work 沒有任何 OP 被 top reply 雙向反駁的 case。

### 1.3 為 OP 所批判的對象辯護
- `S3`：OP 批判 app 自介功利化，`S3.R1`（7）防詐辯護（「好過三幾個月先發現佢無業」）+ `S3.R2`（5）趨勢辯護（「大勢所趨」）——reader 站在 OP 的對立價值。

### 1.4 一致 endorse 並升華（正面敘事）
- `S6`：`S6.R1`（474）文言文祝福「執子之手與子偕老」+ `S6.R2`（188）「對方有回應就係幸福」——無 disagreement，集體升華 OP 的甜蜜。

### 1.5 共鳴（同 work 接住型）
- `S9.R1`（86）：「『自卑到自大』好有同感，App 入邊真係無個正常」——接 OP 的 coined 短語表認同。
- `S1.R1`（1）「已退 apps」、`S1.R2`（0）「唔好玩」——以「退場」共鳴（弱）。

### 1.6 Bookmark / 跟風（零內容）
- `S5.R1/R2`「留友」「留友😂」——收藏行為，不表態。
- `S8.R2`（7）「師兄好忙」——調侃跟風。

## 3. 與 work 對照的關鍵差異（audience 層）

| 維度 | work | love |
|------|------|------|
| reader 對 OP frame | 接受（即使反方向也同 frame） | **校正 / 改 frame**（心態論 / ideology-critique） |
| 雙向反駁同一 OP | 無 | **有**（S4） |
| 為對立價值辯護 | 罕見 | **有**（S3） |
| 正面敘事的命運 | 被諷刺壓制（work S6） | **被擁抱升華**（love S6） |
| credential disqualification | 多（醫生 / employer / 業界） | 弱 / 無明顯 case |
| 高 likes reply 動作 | reframe / disqualify / actionable | endorse（S6 474）/ 校正（S7 190）/ 反駁（S4 130） |
| audience 資料密度 | 厚（~27） | **薄（~12，S2 為 0）** |

**最重要差異：** love 的 reader 會**進入價值辯論**（反駁、校正、ideology-critique），work 的 reader 基本不辯論（接住 / 升級 / 否決出路，但同方向）。這對應 P3：love 有 N2 價值對撞敘事，work 沒有。
