## Stage Contract

### Purpose
從 15 篇 readings + lexicon 自然歸納幾條敘事。**自然幾條就幾條**，不強制 3-5。

### Output
- 每條敘事 = 短名 + story shape（setup → tension → outcome）+ evidence + boundary / 反例 / inversion + 不處理的 hooks
- 結尾列 cross-narrative 訊號分佈
- 不為敘事打分、不排優先級、不打 sentiment

### What this pass does NOT do
- 不命名 audience reaction 模式（P4）
- 不展開 speaker / position 缺席（P5——但會把 hook 移交）
- 不為敘事歸個別 signal 為「主屬」(signals 通常橫跨多條敘事)
- 不引用 Codex northstar

### Discipline
- 每條敘事必須有 boundary case 或 inversion——沒有反例的敘事不夠 robust，重寫
- 引用必須掛 `S#.OP / S#.R#`
- 敘事是 **story shape**（事件結構），不是 proposition（主張），不是 posture（敘述者站位）

## N1 — 投入不再兌現

### Story shape
*Setup*: 發聲者承擔了某種公認的「應該有回報」投入（學位、年資、苦撐、人設經營、儲蓄性工作態度）。*Tension*: 投入完成後市場 / 場合 / 同儕完全沒兌現預期回報。*Outcome*: 發聲者用粗口、戲謔、minimalist 白描或 self-discount 回應，沒有任何敘事提出「下次怎樣才會兌現」。

### Core evidence

- **`S13.OP`** 是 sharpest case：「大家唔好再讀碩士啦。我公司出份 adv 請 asso 做 admin, 人工係 16-20k 港元。收到超過 40 個申請有碩士學位」——招聘者用一句話 + 一個數字證實學位投入失效。`S13.R1` 用 reader-coined「高級興趣班」（P2 §2.2 標記）把碩士去神聖化；`S13.R2` 揭露 employer 端的 anti-master degree 招聘 bias。
- **`S14.OP`** 是 minimalist 極致：四句話，「我有個朋友終於完成 phd 學位，同佢食食下飯，問佢:『辛苦左咁多年，有咩想講？』 佢話:『粗口。』 我靜左。」OP 4000 likes、留言 11——這個 OP-likes / comment 異常比是 P2 §1.4 戲謔詞群的代表。「辛苦左咁多年」這 6 個字就是這條敘事的 sunk-cost 標準措辭。
- **`S7.OP`** 把投入從學位延伸到**人設經營技術**：「諗好 style 去 match 個 vibe，產生幻象令佢以為我高質」——OP 列舉自己投入了多少 stylistic labour，但結局是「我係全檯（截斷）」被識破。同大學、同學院、同專業，階級背景仍然把人區分成「書香後代」vs「基層雜草」。投入轉化失敗的不只是學歷，是**身份本身的可移動性**。
- **`S2.OP`** 是更隱性的同一條敘事：「以前做 Temp 嗰陣明明好活潑、好敢同人溝通」vs 正式員工身份觸發 paralysis——OP 用 Temp 時期當投入累積，但「正式員工」這個身份升級反而 disqualify 過往的功能。**過往 capability 不能轉換為正式身份的能力**。

### Lexicon hook（從 P2 拉）
- 「辛苦左咁多年」`S14.OP` + 「對住同一樣嘢幾年，仲要唔一定做到 positive result」`S14.R2`——sunk-cost vocabulary
- 「高級興趣班」`S13.R1`、「Permanent head Damage」`S14.R1`——reader-coined 去神聖化詞（P2 §2.2 集中在學歷主題）
- 算帳詞群（P2 §1.3）裡的 16-20k / 40 個申請 / 七年 / 多年——量化 sunk cost

### Boundary / inversion
**`S6.OP` 是這條敘事的明顯 inversion**：80 後咖啡豆創業七年，列舉資產證明投入確實兌現（Bentley、啟德天璽、西沙連天台單位）——OP 主張「即使學歷不高，只要用心做事...為夢想奮鬥」。但這個 inversion 立刻被兩個 mechanism 抵銷：(a) OP 結尾「夢裏一定什麼都有」自帶 self-undercut（P1 watchNote）；(b) `S6.R2`「加埋老豆老母比嘅幾千萬碎銀」（2 likes）公開把成功歸因於繼承。**整個 audit 唯一的「投入有兌現」敘事**在文本層自我拆穿並被讀者諷刺。這在 P2 §6.4 已經以詞層觀察出現：「整個 audit 的擴張詞集中在唯一被質疑的訊號」。

### 這條敘事不處理的（移交 P5）
- 沒有任何 signal 主張「投入失效的責任在制度 / 政策 / 學校 / 僱主」——責任歸屬 in absentia
- 沒有任何 signal 提出「下次該怎樣投入才會兌現」——prescriptive 缺席
- 沒有任何 signal 描述「成功兌現的同代人 / 同學」（除 `S6.OP` 被讀者質疑為繼承）

## N2 — 每條出路同步自證偽

### Story shape
*Setup*: 發聲者（OP 或 reader）提出一條 escape ramp（轉行、創業、讀其他學位、找替代工作、考慮非傳統職業、跨地理）。*Tension*: 在同一個訊號內，這條 escape ramp 立刻被同訊號內的另一個聲音否定——通常是 reader 用 credential 或具體數字證偽。*Outcome*: 出口在它被提出的同一個對話空間內關上。

### Core evidence

- **`S12.OP`** 是 sharpest case：35 字 OP 提出唯一 forward option「而家去讀護士好唔好...」，`S12.R1`（2900 likes，全 audit 最高 reply）立刻三件事關門——「千祈唔好讀護士，現在已經飽和，政府準備招聘更多內地護士來港。(我係醫生)」。reply 的 likes 接近 OP 兩倍，且 `S12.R1` 自報醫生身份提升 disqualification 權威。`S12.R2`「去讀護士即係要繼續再失業多兩年（假設你係讀 EN)」技術性補刀。整個訊號 OP 提出出路、reply 用 credential 加事實關閉，**對話完成於關閉狀態**。
- **`S5.OP`** 提出 franchise 茶飲店是否值得——`S5.R1` 用 415 字結構化拆解（加盟費 + 開業 50-70 萬 + 人工 16-18k + Keeta 抽 28-30% + 利潤 12-15% + 冬天跌 15-20%）做有條件 endorsement，但 `S5.R2`「先諗每個月可以蝕幾多」收結。**這條出路沒被完全關閉，但被改寫為「先想能蝕多少」的 risk 計算**——出路保留但 valence 倒轉。
- **`S15.OP`** 是 escape ramp 的極端：「老母講笑咁俾我睇紅磡有殯儀館請看更 30k 一個月問我敢唔敢做 呢一刻我有種衝動 quit u 應徵」——把母親的玩笑接成真實衝動。30k 看更被當作 escape ramp，但它的合理性建立在**逃離當前 trajectory 的衝動**而非職業選擇。Queued 狀態無 reply 資料，下游不能補完，但 OP 本身已經把 escape ramp 包裝成 ambivalence（「敢唔敢做」）。
- **`S11.R2`** 是這條敘事最完整的論述化版本——`S11.OP` 用「全民整水喉」punchline 諷刺黃仁勳的藍領救生圈，`S11.R2`（32 likes）把它升級成 feedback loop：「個個被炒嘅都走去整水喉，競爭激烈，另一方面越來越多人冇收入，屋都冇得住，冇水喉俾你整...結果整水喉都冇得做」。**escape ramp 不是被某個事件關閉，是被它自己的 saturation dynamic 預先關閉**。
- **`S13.OP`** + **`S14.OP`** 也在這條敘事的弱版裡——它們是已經完成投入後的 escape 否決（不要再讀），但這個動作主要屬於 N1 sunk-cost，所以放在邊界。

### Lexicon hook（從 P2 拉）
- P2 §1.1：「飽和」橫跨 4 個 sector 失去 sector specificity——這正是 N2 的 lexical engine。「飽和」變成 generic 是因為**任何被命名的 sector 都會被立刻 saturation-coded**
- P2 §6.3：「希望」零出現、未來時態詞偏向 negative——N2 的 outcome 結構決定了 future-positive vocabulary 沒有立足點

### Boundary / inversion
- **`S3.OP`**：「未轉到工，決定將薪水小偷行動升級」——OP 沒提出真正的 escape ramp，而是把 escape 縮小到日常 protocol 層級（多斟水、多去廁所）。`S3.R1/R2` 接力升級這個微 protocol。**這篇是 N2 的負形**：當 macro escape 全部關閉，micro escape 變成戲謔的代償，且這個微 escape 不會被 saturation 證偽——因為它太小不值得反駁。
- **`S6.OP`**：是 N2 的 inversion，主張「七年前選擇的 escape ramp（從咖啡師到創業）真的走通了」。但同樣被 `S6.R2` 的諷刺 + 自我 self-undercut 抵銷（與 N1 共用 inversion 機制）。
- **`S4.OP`**：「我喺外國返工」是地理 escape，但本文留言區漂移到廁所笑話，沒在這條敘事內被處理——boundary 訊號。

### 這條敘事不處理的（移交 P5）
- 沒有「我提出一個出路，討論之後仍然成立」的訊號
- 沒有 reader 為某條 escape ramp 辯護的 evidence——所有 forward option 都只有 OP 在主張，reader 永遠在關閉
- 「整水喉」「殯儀館看更」這類 micro-occupation 出路被提出的 frequency 暗示某類職業在此議題群中作為「最後的退路」象徵——但這幾個職業裡的真正從業者沒發聲（移交 P5）

## N3 — 經營體面 / 隱藏實情 本身就是工作

### Story shape
*Setup*: 發聲者描述某個職場 / 社交 / 親密關係場景。*Tension*: 真實處境（無能、貧困、失業、出身）與場合要求的呈現之間有距離。*Outcome*: 發聲者用 vocabulary 命名這個距離（露晒餡、薪水小偷、唔敢同老婆講、扮演人設、產生幻象），但沒有解決距離；vocabulary 本身成為勞動產品。

### Core evidence

- **`S7.OP`** 是這條敘事最詳細的版本：OP 完整披露一套 impression-management protocol——「諗好 style 去 match 個 vibe」、「扮演人設提供智性對話/情緒價值」、「產生幻象令佢以為我高質」。最關鍵的不是 OP 經營了人設，是 **OP 把這套經營本身寫成可命名、可分析的勞動**。OP 是元披露——披露的內容是「我如何披露 / 不披露」。
- **`S2.OP`** 用一個粵語短句濃縮同一條：「好驚俾人覺得我『露晒餡』」——「露晒餡」是 vocabulary，工作場域被定性為**保持表層完整性**的場域。`S2.R1/R2` 沒否定這個 framing，而是用 emotional permission（容錯率、預咗死心態）接住，等於同意「保持表層完整性確實是工作」。
- **`S12.OP`** 35 字裡有一句「唔敢同老婆講」——把職場困境的隱瞞**從工作場景延伸到親密關係**。`S12.R1` 用 credential 否決 escape ramp 但完全沒處理這個婚姻層的隱瞞。整篇訊號讀者面對的兩個沒解的事：失業 + 配偶溝通。Reader 的 disqualification 只處理前者。
- **`S3.OP`**「薪水小偷行動升級」是這條敘事的 ironic 變體：OP 把消極怠工 reframe 為主動 rebellion。**真實動作**是 banal 的微逃避，**vocabulary 動作**是創造一個 rebellious identity 來容納這個 banal 動作。這也是一種體面經營——不是對僱主，而是對自己的敘事自我。
- **`S6.OP`** 是同一個 mechanism 的反向 case：列舉資產（Bentley、啟德、西沙）+ 勵志結尾 = 對「我已經做到了」這個身份的表演。OP 自己用 `S6.R1`「早啲瞓，足夠休息好重要」維持 motivational poster 語調——OP 自己在 reply 區繼續經營這個體面。

### Lexicon hook（從 P2 拉）
- P2 §1.2 收縮詞群（露晒餡 / 自卑 / 唔叻 / 唔敢）——這組詞描述的是體面斷裂的瞬間，不是體面本身
- P2 §2.1 OP-coined 詞：「薪水小偷」、「死窮鬼」、「唔敢同老婆講」——OP 自鑄 vocabulary 用以**命名隱藏與被看穿之間的張力**
- P2 §6.4「craftsmanship vocabulary 缺席」——成功只能用資產列舉，不能用 intrinsic 價值描述——體面只能由外觀承載

### Boundary / inversion
- **`S10.OP`** 不太屬於這條：「死窮鬼」是自我命名但 OP 沒在隱瞞，OP 在公開求問如何在 17k vs 1000 蚊請食的衝突裡選擇。S10 更貼近 N2 + N4 的算帳模式。但「本身自己又唔係叻」這句是 self-discount，跟 N3 的「露晒餡」恐懼是同一個 register——所以 S10 在邊界
- **`S14.OP`** 不屬於這條：朋友直接回「粗口」+ OP「我靜左」是反向動作——**拒絕經營體面**，反而把粗口擺出來當答案
- **`S9.OP`** 也不屬於：「仆你街，洗唔洗俾埋個屎忽你吊丫」是公開反體面——OP 不掩飾粗口，是 deliberate 的 register choice

### 這條敘事不處理的（移交 P5）
- 「不經營體面」作為 explicit choice 只出現在 `S14`、`S9`、`S6.R2`——但這些不構成集體 stance
- 沒有任何 signal 描述「老闆 / 同事 / 同學如何看穿這個經營」——表演方的視角單薄
- 「同事 / 配偶 / 家人為什麼需要這個體面」這個問題沒被任何訊號處理

## N4 — 把矛頭指向系統，但不指方向

### Story shape
*Setup*: 發聲者命名一個結構性肇因（外勞政策、AI 取代、稅收結構、平台勞工飽和、政府規劃缺位）。*Tension*: 命名是清晰的、有數據的、甚至帶具體政策對照（新加坡、馬雲預言、渣打 8000 人）。*Outcome*: 沒有任何 signal 從命名升級到 prescription——沒有人主張「應該怎樣做」。**命名動作完成，但 vocabulary 不延伸到 institutional response**。

### Core evidence

- **`S9.OP`** + **`S9.R1`** + **`S9.R2`** 是三層論述接力的代表：OP 用粗口 + hashtag 把職場低薪問題 escalate 到生育議題（「#你哋啲仔女都受緊」「#所以咪無人生 bb 囉」）；`S9.R1` 用 519 字結構分析（新加坡外勞政策對照、地盤鋁模生產力、外勞工資 1/3-1/4、政府規劃缺位）；`S9.R2` 收斂為三字「減人工」。**三層接力同方向但全部停在指認**——沒有任何一層提出 actionable 主張。R1 最接近的 prescription 是「最重要個職位規管」——但只是疑問句結尾，不是主張。
- **`S8.OP`**「香港只有不足一般嘅人口要交薪俸稅...所以大家要覺得自己返工交稅係件好偉大嘅事，養起無數大媽同長者」——OP 把 grievance 轉成 moral status 但**沒提出稅制應該怎樣改、福利分配應該怎樣調整**。`S8.R1`（pinned）+ `S8.R2`（OP 自我轉貼）延伸 reframe 但沒升級為主張。
- **`S11.OP`** 是命名最 data-driven 的版本：列舉馬雲預言、渣打 4 年炒 8000、Meta 8000 凌晨派信、Meta 全球裁員——但每一條都是事實陳述，沒有「政府應該」「行業應該」「個體應該如何回應」的 prescriptive 部分。OP 自己的 punchline「全民整水喉？」是諷刺，不是建議。`S11.R2` 升級為惡性循環論證——但**這個論證的方向是預測 collapse，不是 prevent collapse**。

### Lexicon hook（從 P2 拉）
- P2 §6.2 集體 / 制度 vocabulary 完整缺席（工會、罷工、修法、政策提案、申訴等零出現）——這是 N4 的 lexical signature
- P2 §4.2「命令式句頻繁出現於否定方向」——「不要做 X」遠多於「應該做 X」，整個 lexicon 場沒有 prescriptive direction
- P2 §1.3 算帳詞群——`S11.OP` 的精確數字（15% / 8000 / 4 年 / 9 年前）證明發聲者有能力 quantify 結構，但 quantification 停在診斷層

### Boundary / inversion
- **`S5.R1`** 是 N4 的 inversion：給出非常具體的 actionable prescription（「要做既最好搵 50k 租樓下，同埋頂手鋪再翻新。慳裝修同出牌」）——但這個 prescription 限定於 individual 行動層級，不是 collective / institutional layer。整個 audit 唯一有 actionable 建議的 reply，但 scope 限於「如果你要開 franchise，這樣做」。
- **`S10.R2`**「呢啲文化嘅公司啲人都好撚小氣記仇，如果你短期內唔想轉工，就請咗大家飲嘢算 9 數」——也是 individual-level pragmatic advice，不是改變潛規則的主張
- **`S13.OP`**「大家唔好再讀碩士啦」是命令式但**仍在否定方向**——告訴別人不要做 X，不是主張集體應該做 Y

### 這條敘事不處理的（移交 P5）
- 沒有任何 signal 提出「應該由誰提案 / 推動 institutional response」——actor in absentia
- 沒有政策回應 / 官方立場 / 制度評論——institutional voice 缺席
- 結構性指認 + 個體層 advice 並存，**中間層（行業協會、工會、政黨、媒體評論）完全缺席**——這是 P5 必須處理的 layer gap

## Cross-cutting register：戲謔 / 粗口作為公共可發聲性的閾值

這不是第 5 條敘事，而是穿插在 N1-N4 裡的一種 register。值得 P3 紀錄但**不展開為獨立敘事**，因為它沒有自己的 story shape（沒有 setup → tension → outcome 結構），而是其他敘事的**修辭層**。

戲謔詞群（P2 §1.4）橫跨四條敘事：
- N1 sunk-cost：`S14.OP`「粗口」、`S14.R1`「Permanent head Damage」
- N2 escape collapses：`S11.OP`「全民整水喉？」、`S15.OP`「敢唔敢做」
- N3 體面經營：`S3.OP`「薪水小偷」、`S6.OP`「夢裏一定什麼都有」
- N4 結構指認：`S9.OP`「仆你街」、`S10.R1`「叫佢地食屎啦」

**戲謔 / 粗口的功能：把可說的事擴展到原本不可說的範圍**——它讓「捱完沒兌現」「出路也飽和」「制度卸責」「體面崩潰」這幾種否定性事實**獲得公共可發聲性**。沒有戲謔層，N1-N4 的內容會變成 unspeakable 的 private despair。但戲謔本身**不能升級為主張**——它讓事實可說，但同時也把這個事實鎖定在戲謔 register 裡，無法切換到 prescriptive register。

這個 register 在 P4 audience reaction pass 會以另一個角度重現：reader 如何延續 / 升級 / 不延續 OP 的戲謔 register，是 audience 動作的一個重要維度。
