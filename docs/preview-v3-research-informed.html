What actually changed vs current
Canvas: #ffffff → #f0f1f3 (cool neutral). Cards now contrast against background instead of floating in white void.
Card elevation strategy: Flat+border default. Only brief hero and selected cluster get raised shadow. Everything else = outlined card (M3 pattern). Removes visual noise from too-many-glass-cards.
Post B color: Amber #d97706 → Burnt orange #c2410c. More editorial gravitas, less "warning yellow".
Surfaces: Opaque tints (--accent-a-surface: #eef2ff) instead of transparent rgba. Cleaner on any background.
Bubble map bg: Sunken surface (Atlassian) instead of gradient white. Bubbles pop more against recessed grey.
Support metrics: Ant Design Card.Grid (3 equal cells with borders) instead of scattered pills. Data reads like a dashboard KPI strip.
Engagement table: Proper grid with cell borders (Ant Design) instead of bottom-border-only rows. Column alignment with tabular-nums.
Evidence hover: M3 state layer (4% opacity overlay) instead of background color swap. Subtler, more consistent.
Segmented control: M3 segmented button (filled active) instead of macOS capsule. More tool-like, less system-preferences.
Section rail: Sunken tray (Atlassian) instead of floating glass pill. Anchored, not hovering.
Selected panel: Left accent bar (3px) instead of full-border glow. Focal point is clear without visual noise.
KEPT from current: Bubble visual force, evidence-first structure, technique casebook carousel, dual-color identity, reading-weight headlines (17-18px/800).
