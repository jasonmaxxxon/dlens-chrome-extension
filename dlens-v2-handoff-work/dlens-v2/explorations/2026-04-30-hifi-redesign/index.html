V1 · Editorial

Axis · 編輯氣質優先
Typography
Title / quote 全部 Noto Serif TC，body 用 sans。引語以 italic + 左 hairline accent，像情報雜誌的 pull-quote。讀感是這個 variant 的核心契約。
Filter
Underlined-link 樣式，count 用 mono 小字附隨。沒有 chip outline，整列像一條報紙刊頭。
Card surface
無背景、無 border outline，靠 hairline 分隔。Why-section 標籤改成 italic「為什麼值得留意」而非 mono ALL CAPS。
Trade-off
密度最低，但每張卡片有「停下來讀」的吸引力。適合早上花 5 分鐘讀新一批訊號的 founder / journalist。

V2 · Intelligence Tool

Axis · 操作性與編輯氣質平衡
Typography
Title 用 sans medium-weight。引語仍是 serif 但裝在 paper-1 的 cell 裡，可選可掃。Source ID（SIG-2429 / TPC-091）用 mono 露出。
Filter
Quiet pill chips。Active 整顆填 accent；inactive 只有 hairline outline。
Card surface
有微 shadow + paper-tint 背景，分隔感清楚。Action 區放底部、用「primary 按鈕 + ghost link」的標準對比。
Trade-off
密度中等，可操作性最強。Pair Insights / Brief 預估 token 數這類「工具感」訊息只在 V2 露出。

V3 · Compact Extension-Native

Axis · 密度與 popup 直覺
Typography
整體縮一個刻度。Title 13–15px，引語 11.5–12px 並 1-line clamp。為 Why-section 與 quote-cite 做了無損隱藏。
Filter
橫滑 chip row，背景是 paper-1 rounded-rect。Count 直接拼進 chip（「全部·24」），不另外占行。
Card surface
每張卡片只用底 hairline 分隔，無背景無 padding 浪費。一頁可看 4–5 張卡片，幾乎是 V1 的兩倍。
Trade-off
