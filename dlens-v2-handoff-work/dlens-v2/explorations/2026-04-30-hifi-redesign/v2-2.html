Component inventory · V2.2

28 frames 用到的元件——這份清單是給之後寫 React 元件的對照表，不是 UI 內標示。Mode-aware 是用 CSS variable 切 accent，shell 結構與 atom 不變。

01Shell

Frame
720 × 860 popup canvas，圓角 14、shadow-frame。.product-mode / .topic-mode 在這層套 accent 變數。
Rail
64px 寬，紙感 paper-1 → paper-2 漸層，4 個 stage item，active 用左 2px accent rule。Product：收集 / 分類整理 / 已存 / 設定。Topic：收集 / 初步整理 / Casebook / 設定。
Main
Header 56px + Content 區。Header 含 mode dot（accent + glow）、breadcrumb、右側 quiet-time（mono）。

02Mode header / Page header

page-eyebrow
Serif italic 14、accent-deep。每頁第一行的小 lede。
page-title
Serif 22、weight 500，可斷行。Detail 頁改用 detail-title / detail-headline 26。
page-sub
Serif italic 13、mute。一句話定調，不講功能 spec。
tab-row
Casebook / Topic Detail 用 serif tabs，accent 底線標 active，count 用 mono。

03Filters

filter pill
圓角 999 chips，hairline 邊；active = accent 底 + accent-on 字。F-count 用 mono。
用法
產品端：verdict / 分類；議題端：派系 / 來源 / 歸案狀態。同一個 atom，內容隨 mode 換。

04Signal Intelligence Card

SignalCard.regular
paper-card 底、verdict chip / source ID / 標題 / tags / insight / why（accent-soft 左線）/ 引語（accent 左線、paper-card-2 底）/ action row。
SignalCard.hero
本週主推。paper-card-2 漸層、accent 頂線 2px、右上 hero-flag（mono · gold）、22px 標題、shadow-hero。
SignalDetail (full takeover)
取代列表的整頁。detail-headline 26 + 對產品的可能用處 / 為什麼值得留意兩段 prose + quote-hero（❝ 飾）+ 三鈕 row。
PromptRequest (single-card)
linear gradient 底 + 3px accent 左線 + serif italic input。Optional 給 brief 帶問題。

05Saved row + ActionBar

SavedRow.idle / .selected
idle 是 hairline 分隔的列。selected 加 paper-card-2 底、左 2px accent rule、圓角 8。Checkbox 17×17、accent 填色為 on。
ActionBar (floating)
有 selection 時從底部 24px 內縮升起。「已選 N 篇」serif 大字 + accent italic、token 估算 mono、清除 ghost、匯出 primary。
Toast (saved confirmation)
頂端短暫 8s 留存，ink 底 + accent check + undo link。不阻塞列表。

06Export drawer + Brief preview

ExportDrawer
Bottom sheet。兩張 export-mode 卡並排：原文優先（含 12 quote）/ 精簡決策（recommended，accent 邊 + accent-on CTA）。
BriefDoc
Render preview。Serif body、quote 是 italic small、ul 列點為「下一步建議」。複製／下載 .md 兩個 CTA。

07Topic card · Casebook

TopicLead
Casebook 第一篇大版：dateline pulse 點 + 標題 + lead-reading（serif italic emphasis）+ marker quote + lead-bubbles。
TopicCard.condensed
後續議題：grid 1fr / 130，左側 dateline-italic + serif 標題 + reading 2-line clamp + marker，右側 bubble-preview（4 種尺寸 + caption）。
SuggestCard (T2)
初步整理用：linear gradient 底 + 2px accent top、sg-eyebrow（信心數字）、sg-thumbs（候選來源 chips）、sg-actions 三鈕（建立 / 加入 / 先放著）。
InboxRow
raw signals 列表：mono 編號 / serif headline / italic quote / mono meta / 右側狀態（未歸 / 建議歸案）。

08Cluster card · Topic Detail

ClusterToken
main reading 內可點 token：accent-soft 底 + 1.5px accent 底線 + mono C# + ↗ 飾。Hover 翻轉 accent / accent-on。
ClusterBubblesRow
Detail 頁的水平 bubble row。lead 大圈 accent 填、其餘小圈 accent-soft + 1px accent border。
ClusterSection.idle / .focused / .dimmed
idle 是 hairline 分隔；focused = paper-card 底 + accent-softer tint + soft accent ring；dimmed = opacity 0.45。
EvidenceRow.idle / .selected
paper-card 底、圓角 8。selected 加 1.5px accent ring。＋ 加入對比 → 已選 A（accent 填色 chip）。

09Evidence drawer

Drawer (bottom sheet)
圓角 18 頂部、handle bar、drawer-head（標題 + 計數 + 排序 chip group）、drawer-body 列表。Backdrop dim 32%、底層 opacity 0.55。
EvidenceItem
serif italic quote + mono meta + actions（加入對比 / 查看原文）。
SourceListRow
原文清單視角：左 thumb（平台縮寫）、中標題 + 引用 2-line clamp + meta、右 cluster pill + 引用次數。

10Pair tray + Pair Insight

PairTray (floating pill)
底部 20px 浮起圓角膠囊。狀態：選 1 → prompt「再選一篇⋯⋯」+ disabled 對比鈕；選 2 → 兩 chip + active 對比鈕。dismiss ✕。
PairInsightCard
短保存態：conclusion 15.5、A/B 半邊框架（mono label + serif body）、amplify-to-cluster row（cluster-token 連結）、saved-flag + 時間 + 編輯／展開完整對比。
PairInsightCard.hero
選 2 時 raised 預覽 / 草稿編輯態。padding 22 + shadow-hero + 2px accent top。conclusion 18。

11Settings group

SettingsGroup
paper-card 底、serif h4 + sg-desc italic、sg-row（label + value/toggle）。
ContextText (Product)
paper-0 內框 + hairline 邊 + serif italic body + placeholder mute。Markdown 風格的 PRD slot。
FileTile (Product)
32 icon 縮寫 + 檔名 / 大小 mono / 移除 link。最後一格是上傳 placeholder。
WorkspaceCard (Topic)
1.5px hairline / accent 邊框，當前 workspace 加 accent ring。serif name + badge（當前）+ italic desc + mono meta。

12Collect preview

CollectCanvas
grid 1fr / 240。左側是 collect-target 卡（主要引用 + 同串延伸），右側 collect-side 三張 side-card。
CollectTarget
paper-card 底、ct-eyebrow mono、serif headline、italic quote（accent-soft 左線）、mono meta row。
SideCard
radio list 模式。Product mode：Verdict / 分類 / 同串設定。Topic mode：議題候選 / 派系候選 / 同串設定。
CollectCTA
primary（收進⋯⋯）+ secondary（編輯引用語）+ 右側 quiet 預留 ID。
