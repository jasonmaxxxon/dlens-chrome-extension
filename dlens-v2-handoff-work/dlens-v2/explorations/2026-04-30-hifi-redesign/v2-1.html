Component hierarchy

這次只有一個視覺方向。底下是把 12 frames 用到的元件分組記下來——是給之後寫 React 元件時的對照表，不是 UI 內的標示。

AShell

Rail
64×860，紙感 paper-1 → paper-2 漸層，stage-based 4 個項目，active 項用左 2px accent rule + 紙感 paper-0 板。文案隨 mode 切換（產品：收集／分類整理／已存／設定）。
Header
56px，含 mode dot（accent + glow）、breadcrumb、右側 quiet-time（mono）。Detail 頁加 ← 返回 前綴。
Mode 上下文
不在 popup 內做 mode 切換，只在 settings。Header 用 quiet 文字「DLens · 產品觀察 / 議題觀察」承載 mode context。

BSignal Card 系列

SignalCard.regular
paper-card 底 + soft shadow，含 verdict chip / source ID / 標題 / tags / insight / why（accent-soft 左線）/ 引語（accent 左線、paper-card-2 底）/ action row。
SignalCard.hero
本週主推，加上 paper-card-2 漸層、accent 頂線、右上 hero-flag（mono · gold）、22px 標題。
SignalDetail（full takeover）
取代列表的整頁狀態，含 detail-headline 26px、對產品的可能用處 / 為什麼值得留意兩段 prose、quote-hero 帶 ❝ 飾、底部 detail-actions 三鈕（保存／加入對比／查看完整證據）。
EvidenceDrawer
底部 sheet（圓角 18px、handle bar、drawer-head 標題 + 計數 + 排序、drawer-body evidence-item 列表）。背景 backdrop dim 32%，主畫面 opacity 0.55。

CSaved + Export

SavedRow.idle / .selected
idle 是純 hairline 分隔的列。selected 加 paper-card-2 底、左 2px accent rule、圓角 8px。同一頁混搭 5–6 列。
ActionBar (floating)
當有 selection 時從底部 24px 內縮升起。含「已選 N 篇」（serif 數字 + accent italic）、token 估算（mono）、清除（ghost）、匯出 Brief（primary）。
ExportDrawer
Bottom sheet，內含兩張 export-mode 卡並排。右側「精簡決策 Brief」帶 recommended（accent 邊 + accent-on CTA）。
ExportMode
